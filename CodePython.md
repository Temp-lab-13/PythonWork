## Списки

> Список
>> Это упорядоченныйконечный набор элементов. По сути это тот же массив, в котором можно хранить элементы любого типа данных.

__Создание списка__
```python
# Создание путого списка.
list_a = [] 
# Создание пустого списка 2.
list_b = list() 
# Создание списка с заданными первичными элементами элементами.
list_c = [7, 4, 1, 0] 
```
__Обращение к элементу списка__
```python
# Нумерация списка происходит как и у массива с 0.
list_c = [7, 4, 1, 0] 
# Вывод первого элемента списка.
print(list_c(0)) # [7]
# Вывод последнего элемента.
# при использовании `-`, инексация списка начинается с конца, а с чёт с -1.
print(list_c(-1)) # [0]
# Вывод всех элементов списка.
print(list_c) # [7, 4, 1, 0]
# * - убирает при выводе списка скобки и запятые.
print(*list_c) # 7 4 1 0
```
> Перебор элементов списка с помощью цикла `for`.
```python
list_c = [7, 4, 1, 0] 
for i in list_c:
    print(i)
# Цикл будет по очерёдно проходить по элементам списка и распечатывать их`(i)`.
```
> Длинну(размер) списка можно узнать с помощь команды `len`.
```python
list_c = [7, 4, 1, 0] 
print(len(list_c)) # 4
```

__Добавление  и удаление эелемента в список__

> Функция `append` позволяет добавить элемент в конец списка.
```python
list_c = [7, 4, 1, 0] 
print(list_c) # [7, 4, 1, 0] 
list_c.append(8)
print(list_c) # [7, 4, 1, 0, 8] 
```
> Функция `insert(x, x)` позволяет добовлять елементы на уже конкретную позиция в списке.

Первый аргумент обозначает индекс(место в списке) по которому добавляется элемент в список. Второй аргумент - непосредственно добовляемый элемент. При этом, все имеющиеся в списке элементы, сдивигаются в право от вставляемого элемента.
```python
list_c = [7, 4, 1, 0] 
print(list_c) # [7, 4, 1, 0] 
list_c.insert(1, 22)
print(list_c) # [7, 22, 4, 1, 0, 8] 
```

> Функция `pop()` позволяет удалить последний элемент списка. Так же эта функция возращает удаляемый элемент, что позволяет распечатать его или положить в отдельную переменную.

Так же, функция позволяет задать в аргументах индекс конкретного элемента, и удалить его.
```python
list_c = [7, 4, 1, 0] 
# Удаление последнего элемента
print(list_c) # [7, 4, 1, 0] 
list_c.pop()
print(list_c) # [7, 4, 1] 
# Удаление и распечатывание удаляемого элемента.
print(list_c.pop()) # 1
print(list_c) # [7, 4]
# Присвоение удаляемого элемента в отдельную переменную.
f = list_c.pop()
print(f) # 4
# Удаление элемента по индексу.
print(list_c) # [7, 4, 1, 0] 
list_c.pop(1) # Удаляется второй элемента `4`
print(list_c) # [7, 1, 0]
```

## Кортеж
> Кортеж - неизменяемый список.

__Создание кортежа__

```python
# Создание путого кортежа.
a = ()
print(type(a)) # class <`tuple`>
# Создание кортежа с прописанными данными.
a = (1,)
print(a) # (1)
# кортежу так же можно присвоить список.
a = [1, 11, 1990]
c = tuple(a)
print(c) # (1, 11, 1990)
```

__Разделение кортежа__

Элементы кортежа так же можно разделить и присвоить разным переменным.
```python
a = (1, 11, 1990,)
c,v,b = a
print(c) # 1
print(v) # 11
print(b) # 1990
```

## Словари

> Словари 
>> это не упорядаченная колекция произвольных объектов с доступо по ключу.

__Создание словаря__
```python
a = {} # Создание пуксмтого словаря.
s = dict() # Создание пуксмтого словаря.

```
__Добавление в словарь элемента и ключа.__

```python
a = {} 
a[`q`] = `qwerty`
# Где `q` - это ключь, по которому мы можем достать добавленный элемент.
# А `qwerty` = уже сам добовляемый элемент.
ф = {'q': 'qwerty'} # Алтернатива. Создание словаря с имеющимися данными.
```

__Обращение к словарю или элементу__

```python
a = {} # Создание пуксмтого словаря.
a[`q`] = `qwerty`
# Вывод всего словаря и ключей к его элементам.
print(a) # {`q`: `qwerty`}
# Вывод конкретного элемента по его ключу.
print(a[`q`]) # qwerty
```
Функция `items` позволяет каждый элемент и его ключь преобразовать в кортеж. И тем самым возращать как отдельные элементы (для разбиение по переменным, к примеру.)

```python
a = {'q': 'qwerty'}
print(a.inems) # [(`q`, `qwerty`)]
s, d = a
print(s) # q
```
__Удаление элемента__
Удалить элемент из словаря можно с пмощью команды `del`
Для этого необхоимо обратится к к ключу элемента который нужно удалить.
```python
a = {'q': 'qwerty'}
del a['q']
```

## Множества
> Множество
>> это список уникальных значений.

__Создание множества__
```python
a = {'значение1', 'значение2'}
a = set() # Пустое множество
```

__Добавление в множество__
```python
a = {'значение1', 'значение2'}
a.add('значение3')
```

__Удаление из мнодества__
```python
a = {'значение1', 'значение2'}
a.remove('значение2')
# Следующая функция перед удалением проверяет наличие удаляемого элемента, что позволяет избежать ошибок.
a.discard('значение2')
# Способ полностью очистить множество.
a.clear()
```

__Операции со множествами__
```python
с = a.cope() # копировать множество `а` в мнодество `c`
u = a.union(b) # Обединить множества а и b в u
i = a.intersection(b) # объединить только пересекающиеся(одинаковые) елементы двух мнодеств а и b присвоив их i
x = a.difference(b) # найти разность множеств, в х сохраняться только те элементы из а, которых нет в b
v = b.difference(a) # и тоже самое, но только в v  созранятся те элементы из b, которых нет в а.
# Команды могут совмещаться.
n = frozenset(a) # Замораживает множество. Т.е. множество присвоенное b, нельзя будет никак изментиь.
```
